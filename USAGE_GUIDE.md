# Usage Guide

## Step1:

1. Turn on the YuMi robot and the power supply.
2. Open **Terminal 1**:
   ```bash
   mamba activate cable
   yumi # source the ws
   roslaunch zed_wrapper zedm.launch
    ```
3. Open **Terminal 2**:
   ```bash
   mamba activate cable
   yumi # source the ws
   rviz
    ```
